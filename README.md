# 财务分析自动化脚本 📊

会计专业学生做的第一份 Python 数据分析作品。

## 它能干什么

运行后自动完成以下工作（无需任何操作）：
- 读取三家公司（制造/零售/科技）的财务数据
- 计算每个公司的完整杜邦分析：ROE = 净利率 × 资产周转率 × 权益乘数
- 生成横向对比表格
- 自动生成中文解读（ROE驱动因素分析）
- 输出到控制台 + 自动保存报告文件

## 怎么运行

### 第一步：装 Python
1. 打开 https://www.python.org/downloads/
2. 下载最新版（3.12或3.13）
3. 安装时勾选 "Add Python to PATH"（重要！）
4. 打开终端（Win+R → 输入 cmd → 回车），输入 python --version，看到版本号即成功

### 第二步：运行脚本
python 财务分析自动化.py
不用按任何按钮，输出结果会直接显示在屏幕上，同时自动在文件夹里生成 财务分析报告.txt

## 我能用它学什么

- 看懂了 calc_ratios() 函数 → 你就看懂了杜邦分析的代码实现
- 把里面的三家公司数据改成自己想分析的公司 → 你就学会了改代码
- 给脚本加一个新的指标（比如流动比率） → 你就学会了自己动手完善
